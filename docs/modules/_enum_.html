defaultNext

  • defaultNext<A>(toEnum: (n: Int32) => Option<A>, fromEnum: (a: A) => Int32): (a: A) => Option<A>
    • Defined in Dev/fp-ts-numerics/src/Enum.ts:34
    since

    1.0.0

    Provides a default implementation for next, given a function that maps integers to values in the Enum, and a function that maps values in the Enum back to integers. The integer mapping must agree in both directions for this to implement a law-abiding next.

    If a BoundedEnum instance exists for a, the toEnum and fromEnum functions can be used here:

    next = defaultNext(toEnum, fromEnum)
